# 一款工具搞定所有 AI智能体助手，我再也不用在十几个AI工具之间来回切了

你是不是也有过这样烦恼：电脑上装了一堆 AI 工具，Kimi、豆包、ChatGPT、Gemini……用的时候要来回切换，麻烦得要命。

我自己作为一个重度 AI 用户，每天都在用好几个不同的 AI智能体助手，切换来切换去太烦了。

后来我找到了一个解决方案——Cherry Studio，一款能把所有 AI智能体助手 整合到一个界面的神器。

## Cherry Studio 是什么

一句话概括：免费开源的 本地AI 聚合工具。

- **支持平台**：Windows / macOS / Linux

- **官网**：cherry-ai.com
  
![[7.公众号/附件/官网介绍.png]]  


Cherry Studio 支持两种方式：

**免费方式**：小程序（网页版 AI）

我常用的gemini、豆包、Kimi 直接固定在左侧工具栏。简单问题用这个，直接登录你之前的账号, 数据都在, 且不花钱。

![[7.公众号/附件/使用gemini.png]]

国内外常用的AI工具这里都支持:
![[7.公众号/附件/小程序.png]]
 

**付费方式**：API 配置

复杂问题我用 DeepSeek-V3.2 / 智谱GLM-4.7 官方的 API，性价比超高：
  
![[7.公众号/附件/deepseek api配制页.png]]


## 三步搞定配置

### 第一步：获取 API Key

去 DeepSeek 官网注册，申请一个 API Key。
![[7.公众号/附件/deepseek开放平台.png]]
  
### 第二步：在 Cherry Studio 里配置


打开设置，找到"模型服务"，把 API Key 粘贴进去，然后添加模型。

![[7.公众号/附件/deepseek api配制页 1.png]]
  

### 第三步：添加智能体助手


Cherry Studio 内置了上千个智能体，产品经理、程序员、文案写手……随便挑。

  ![[7.公众号/附件/助手库.png]]

## 成本透明

DeepSeek / 智谱 作为国产之光,  API 很便宜：
- deepseek-chat：约 0.001 元/千 tokens
- zhipuai-glm-4.7：最新出的glm-4.7(能力对标claude-4.5), 包月只要20块. 比直接用 Claude/ ChatGPT那些国外模型 省钱多了。
