# 一款工具搞定所有 AI，我再也不用在十几个 App 之间来回切了

人就是有一种和自己过不去的瘾。

你电脑上是不是装了一堆 AI 工具，Kimi、豆包、ChatGPT、Gemini、DeepSeek......每天像个神经病一样在十几个网页之间切来切去，烦不烦？我看着都累。

![多个AI工具混乱场景](images/zhihu_2.jpg)

说白了这就是典型的工具焦虑，好像装得越多就越牛逼似的，其实本质上就是懒得找解决方案。

后来我发现了个叫 Cherry Studio 的玩意儿，免费开源，能把所有 AI 整到一个界面里，问题迎刃而解。

![Cherry Studio 界面](images/cherry_studio_homepage.png)

这软件有两种用法。

第一种是小程序，说白了就是把网页版的 AI 嵌进来，豆包啊 Kimi 啊直接固定在左侧工具栏，简单问题用这个，不花钱。

![Cherry Studio 左侧小程序](images/zhihu_5.jpg)

第二种是配 API，复杂问题我就用 DeepSeek 官方的 API，deepseek-reasoner 搞深度推理，deepseek-chat 快速输出，性价比高得离谱。

![模型配置界面](images/zhihu_6.jpg)

配置起来也简单，三步搞定。

第一步去 DeepSeek 官网搞个 API Key，第二步在 Cherry Studio 里填进去，第三步添加模型。

![DeepSeek 官网](images/deepseek_platform.png)

![Cherry Studio API 配置](images/zhihu_8.jpg)

就这么简单，Cherry Studio 里还有上千个智能体，产品经理、程序员、文案写手，随便挑。

![智能体库](images/zhihu_11.jpg)

用了半年多，这玩意儿已经是我主力生产力工具了，一个软件搞定所有 AI，不用来回切，还支持导出到 Notion、Obsidian。

![导出功能](images/zhihu_14.jpg)

成本也很透明，DeepSeek 的 API 便宜得离谱，deepseek-chat 才 0.001 元每千 tokens，比 ChatGPT 省钱多了。

且慢，有人要说了，我这么折腾图个啥？

不图啥，就是不想和自己过不去了。

![总结](images/zhihu_15.jpg)
