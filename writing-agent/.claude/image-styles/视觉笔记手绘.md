# 视觉笔记手绘风格

## 提示词模板

#角色
你是一位专业的视觉笔记艺术家和信息设计师。

#任务
根据我提供的文本内容，创建一个清晰、简洁的视觉笔记风格手绘草图图像，旨在帮助读者快速掌握其内在逻辑和核心要点。

#步骤
- 阅读并提取"关键节点"（角色/行动/结果/条件），按流程或因果关系进行组织；
- 使用简洁的英文关键词为关键节点命名，确保拼写正确且易于理解；
- 按照#核心要求创建图像；
- 使生成的效果更加简洁。

#核心要求
- 视觉风格：严格遵循视觉笔记手绘风格。所有元素都应具有手写感。使用干净、极简的线条，搭配1-2个简单的图标。整体感觉应像用黑色记号笔在白板或笔记本上绘制。
- 构图与布局：整体布局必须清晰、简洁且逻辑性强，自然引导读者的视线。整体布局不应严格遵循从左到右或从上到下的顺序；合理自由地安排元素，确保元素之间有足够的留白。严格避免箭头重叠；确保视觉效果不杂乱。
- 颜色：仅使用黑色和白色，保持整体简洁性；纯白色背景，高对比度；无渐变、无阴影、无照片/3D/拟物化效果。

#输出目标
生成一个极简的手绘草图，解释原始文本的核心思想，让任何看到图像的人都能快速掌握主要要点。

我提供的内容是: {description}

## 参数设置

- 模型: nano-banana-pro
- 图片格式: png
- 质量: high

## 使用说明

在调用生成图片时，将模板中的 `{description}` 替换为用户预留的图片描述内容。

示例：
```
用户预留描述: "Claude Code是一个AI助手工具，就像你的私人秘书"

最终提示词: [上述模板] + "Claude Code是一个AI助手工具，就像你的私人秘书"
```

## 适用场景

- 概念解释图
- 流程图解
- 知识点总结
- 逻辑关系展示
