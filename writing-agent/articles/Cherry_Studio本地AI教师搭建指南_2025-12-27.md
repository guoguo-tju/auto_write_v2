## 开始搭建，只需要 3 步

  

### 步骤 1：安装 Cherry Studio

  

这个蛮简单的。

  

去 Cherry Studio 官网下个安装包就行：https://cherry-ai.com

  

安装完打开，左下角有个"设置"按钮，点进去就能配置了。

![[7.公众号/附件/官网介绍 2.png]]
  

### 步骤 2：接入智谱 GLM-4.7

  

这一步是核心，我们要让 Cherry Studio 能调用智谱 GLM。

  

**2.1 注册智谱开放平台并获取 API 密钥**


1. 访问智谱 AI 开放平台：https://www.bigmodel.cn/glm-coding?ic=1HVEGRP0BR , 用最低档20元/月即可

2. 注册/登录账号，完成实名认证

3. 进入"API 密钥"页面，点击"新建 API Key"

4. 复制生成的密钥（格式像一串随机字符）

  

**2.2 在 Cherry Studio 中配置智谱 GLM**

  

1. 打开 Cherry Studio，点击左下角"设置" → "模型服务" → "Anthropic"

2. 填写配置：

- API 地址：https://open.bigmodel.cn/api/anthropic

- API 密钥：粘贴刚才复制的密钥

- 模型 ID：`claude-sonnet-4-5` (支持图片识别, GLM专门对claude code做的优化, 背后是GLM-4.7)

3. 点击"检查"验证连接，显示"连接成功"就搞定了
![[7.公众号/附件/图2.png]]
  


这样配置完，Cherry Studio 就能用 GLM-4.7 模型了(且支持图片识别)，智谱最新款，推理能力强。

  

### 步骤 3：接入 PromptX MCP（关键步骤）

  

这一步是重点，PromptX 是基于 MCP 协议的 AI 角色和记忆平台，能让 AI 教师具备专业教学能力和长期记忆。

  

**3.1 在 Cherry Studio 中配置 MCP 服务器**

  

1. 打开 Cherry Studio，点击左下角"设置" → "MCP 服务器"

2. 点击新增"编辑 JSON"，添加以下配置：



```json
{
  "mcpServers": {
    "promptx": {
      "command": "npx",
      "args": ["-y", "@promptx/mcp-server"]
    }
  }
}
```

  

3. 保存后，点击"搜索 MCP"，找到 `promptx` 并添加

4. 显示绿色对钩就说明连接成功了
![[7.公众号/附件/图4.png]]
  
  

**3.2 用女娲创建专属 AI 教师角色**

  

PromptX 有个神奇功能叫"女娲"，能通过自然语言描述在 2 分钟内创建专属 AI 角色。

  

在对话界面输入："激活 nuwa" , 激活女娲角色创造顾问。

  

然后描述你的需求，比如：

  

"我需要一个初中语文教师，擅长作文批改。能指出文章结构问题、表达不当的地方，给出具体修改建议，并生成范文参考。"

  

女娲会：

1. 理解你的需求并不断地追问你来确认细节
![[7.公众号/附件/图5.png]]

2. 自动生成符合标准的 AI 教师角色
![[7.公众号/附件/图6.png]]

3. 返回角色 ID（比如 `chinese-teacher`）, 然后输入 '激活chinese-teacher' 就能激活使用了
![[7.公众号/附件/图7.png]]

  



整个过程不到 2 分钟，比你手动写提示词专业多了。

  

## 知识点：女娲 vs 手动配置

  

你可能会问，女娲和我手动写提示词有啥区别？

  

**女娲的优势：**

  

1. **零门槛**：用自然语言描述就行，不用学复杂的语法

2. **够专业**：生成的角色符合 DPML 标准 比手写的更规范, 并且不断追问深度挖掘你的个性化需求

3. **速度快**：2 分钟搞定，手动写提示词可能要半小时

4. **可复用**：生成角色 ID 后，随时可以激活，还能分享给别人用

  
  

## 实际用起来效果怎么样？

## 避坑指南

  

最后说下我踩过的坑，你注意下能少走弯路：

  

**坑 1：女娲激活没反应**

  

输入 `promptx_action nuwa` 没反应？先确认 MCP 服务器已经连接成功（显示绿色对钩）。有时候 npx 首次运行会下载包，需要等一会儿。

  

**坑 2：女娲理解错需求**

  

女娲虽然聪明，但有时候会理解偏差。建议描述需求的时候尽量具体，比如：

- "我需要初中语文教师" → "我需要初中语文教师，擅长作文批改和阅读理解指导"

- 多给点场景细节，女娲生成的角色会更精准

  

**坑 3：MCP 服务器连接不上**

  

配置完 JSON 后如果找不到 promptx，检查一下命令是不是写对了。JSON 格式要严格，少个逗号都不行。
